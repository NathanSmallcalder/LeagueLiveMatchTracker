About This Project

This is a live League of Legends win probability tracker built as a CV project demonstrating:

  • PyTorch deep learning - Neural network trained on 1.7M+ match intervals
  • Real-time data ingestion - Connects to Riot's live client API
  • Flask web app - Hextech-themed UI with live Chart.js updates
  • 1821 ML features - Gold, XP, CS, kills, champion matchups, objectives
  • Docker deployment - Containerized and ready to run anywhere

Technical Stack

Backend: Python, Flask, PyTorch, Pandas, Requests

Frontend: HTML/CSS/JS, Chart.js, Hextech-themed design

Data: Riot API (live client), Data Dragon (item data)

Deployment: Docker, Dockerfile included

ML Features (1821 columns)

Team Features (Match Level)

  • team_kills (differential)
  • team_inhibitors
  • team_towers
  • team_dragons
  • team_barons
  • team_void_grubs
  • team_heralds
  • at_soul_point

Player Stats (Per Slot 0-9)

  • total_gold_0 to 9
  • xp_0 to 9
  • level_0 to 9
  • kills_0 to 9
  • deaths_0 to 9
  • assists_0 to 9
  • cs_0 to 9
  • jungle_cs_0 to 9

Lane Differentials

  • gold_diff_lane_0 to 4
  • xp_diff_lane_0 to 4

Champion One-Hot Encoding

~1600 binary columns (champion_{name}_{slot})

Model Architecture

- Input: 1818 features

- Layer 1: 1024 neurons (BatchNorm, ReLU, Dropout 0.6)

- Layer 2: 256 neurons (BatchNorm, ReLU)

- Output: Sigmoid (Red Win Probability)
